# 🚀 100 天搞定 Agent 开发

> **一条从理解 LLM 本质，到构建可控 Agent 系统的工程化学习路径**

**2025 年，是 Agent 真正开始爆发的一年。**
Agent 不再只是 Demo 和 Prompt 展示，而是开始进入真实业务系统。

我是一个 **Agent 开发者**，已经将 Agent 落地到实际项目中，也在过程中踩过大量坑：
Prompt 失控、上下文膨胀、系统不可观测、复杂度失衡……

基于这些真实经验，我在实践中结合 AI 一起梳理和修正认知与表达，逐步沉淀出了这个 repo：
一方面作为自己的**工程总结**，另一方面希望能为走在 Agent 开发道路上的工程师**减少试错成本**。

## 👥 适合谁

- 有编程基础，想系统性进入 **Agent / AI 工程** 的开发者
- 用过 LLM / LangChain，但感觉「越写越乱」的人
- 关心 **可维护、可演进 Agent 架构** 的工程师 / 架构师

## 🛠️ 环境配置

> 本教程文档，默认运行在 **Python + Jupyter Notebook** 环境中

### 基础环境要求

- **Python**：`>= 3.12`

### 1. 安装依赖并配置环境变量

a.安装 uv

```bash
pip install uv
```

b.在项目根目录执行按照依赖：

```bash
uv sync
```

c.环境变量配置

```env
cp .env.example .env
```

在 .env 文件中配置自己的 API Key

### 2.启动 Jupyter Notebook

```bash
uv run jupyter notebook
```

成功启动后打开浏览器进行访问即可，默认地址为 `http://localhost:8888`
